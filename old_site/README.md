# ========================================
# СТАРЫЙ САЙТ - Joomla CMS
# ========================================

Сюда можно скопировать любые заметки о старом сайте.

## Файлы которые нужно скопировать:

### Обязательно:
- [x] configuration.php — ПАРОЛЬ ОТ БД

### Желательно (если есть кастомные компоненты):
- [ ] components/com_* — кастомные компоненты
- [ ] administrator/components/com_* — админ части компонентов

## Структура Joomla:

```
public_html/
├── administrator/     — Админка Joomla
├── components/        — Компоненты фронтенда  
├── modules/          — Модули
├── plugins/          — Плагины
├── templates/        — Шаблоны
├── configuration.php — КОНФИГ С ПАРОЛЯМИ
└── index.php         — Точка входа
```
